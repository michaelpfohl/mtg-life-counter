# Magic: The Gathering Life Counter

App that lets you keep track of the life totals of any number of players while playing Magic: The Gathering. 

## Features: 
- Add Player
- Rename Player
- Remove Player
- Increment life total by 1 or 5
- Reset life total
- Default life totals to 40 by enabling Commander mode
- Default life totals to 20 by enabling Standard Mode
